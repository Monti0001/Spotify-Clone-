# 🎵 Spotify Web Player Clone

A responsive Spotify Web Player UI clone built using HTML5 and CSS3.  
This project focuses on recreating Spotify’s user interface for learning and practice purposes.

## ✨ Features
- Sidebar navigation (Home, Search, Library)
- Sticky navigation header
- Music cards layout (Recently Played, Trending, Charts)
- Fixed bottom music player with controls
- Responsive layout using CSS Flexbox
- Clean and modern UI inspired by Spotify

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox)
- Font Awesome Icons
- Google Fonts (Montserrat)
